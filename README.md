 Explique o que é o try-catch e como ele se conecta com a Prevenção de Erros.

 O try-catch é uma estrutura usada para tratar exceções em um programa. Ele permite capturar erros que podem ocorrer durante a execução e evitar que o sistema seja interrompido, contribuindo para a prevenção de falhas e melhorando a experiência do usuário.
